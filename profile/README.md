# Note to Developers

### Membuat Proyek Nuxt

Jika Anda ingin membuat proyek baru menggunakan **Nuxt**, wajib menggunakan template dari repository resmi kami:

**Repository Template:** [https://github.com/sadigitID/nuxt-template](https://github.com/sadigitID/nuxt-template)

#### Command untuk Membuat Proyek Nuxt

```bash
npm create nuxt@latest -- -t github:sadigitid/nuxt-template
```

Template ini sudah dilengkapi dengan berbagai aturan dan petunjuk untuk developer yang harus diikuti dalam pengembangan proyek.

### Membuat Proyek NestJS

Jika Anda ingin membuat proyek baru menggunakan **NestJS** (Backend/REST API), wajib menggunakan template dari repository resmi kami:

**Repository Template:** [https://github.com/sadigitID/nest-template](https://github.com/sadigitID/nest-template)

#### Command untuk Membuat Proyek NestJS

```bash
npx degit sadigitid/nest-template nama-project
```

Template ini sudah dilengkapi dengan berbagai aturan dan petunjuk untuk developer yang harus diikuti dalam pengembangan proyek. Termasuk: modular architecture, Swagger/OpenAPI documentation, validation DTO, global exception filter, rate limiting, unit & E2E testing, dan Docker setup.

---

**Catatan Penting:**
- Semua template tersebut berisi standar coding, konfigurasi, dan best practices yang telah ditetapkan oleh tim.
- Pastikan untuk membaca dokumentasi di masing-masing repository template sebelum memulai pengembangan.
- Jangan mengubah konfigurasi dasar yang telah disediakan tanpa persetujuan tim.
