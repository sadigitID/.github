# Note to Developers

### Membuat Proyek Nuxt

Jika Anda ingin membuat proyek baru menggunakan **Nuxt**, wajib menggunakan template dari repository resmi kami:

**Repository Template:** [https://github.com/sadigitID/nuxt-template](https://github.com/sadigitID/nuxt-template)

#### Command untuk Membuat Proyek Nuxt

```bash
npm create nuxt@latest -- -t github:sadigitid/nuxt-template
```

Template ini sudah dilengkapi dengan berbagai aturan dan petunjuk untuk developer yang harus diikuti dalam pengembangan proyek.

### Membuat Proyek Vue + Vite

Jika Anda ingin membuat proyek baru menggunakan **Vue + Vite**, wajib menggunakan template dari repository resmi kami:

**Repository Template:** [https://github.com/sadigitID/vue-template](https://github.com/sadigitID/vue-template)

#### Command untuk Membuat Proyek Vue + Vite

```bash
npx degit sadigitid/vue-template nama-project
```

Template ini sudah dilengkapi dengan berbagai aturan dan petunjuk untuk developer yang harus diikuti dalam pengembangan proyek.

---

**Catatan Penting:**
- Kedua template tersebut berisi standar coding, konfigurasi, dan best practices yang telah ditetapkan oleh tim.
- Pastikan untuk membaca dokumentasi di masing-masing repository template sebelum memulai pengembangan.
- Jangan mengubah konfigurasi dasar yang telah disediakan tanpa persetujuan tim.
