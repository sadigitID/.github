# Note to Developers

Gunakan template resmi untuk membuat proyek baru.

## Nuxt
```bash
npm create nuxt@latest -- -t github:sadigitid/nuxt-template
```
[sadigitID/nuxt-template](https://github.com/sadigitID/nuxt-template)

## Vue + Vite
```bash
npx degit sadigitid/vue-template nama-project
```
[sadigitID/vue-template](https://github.com/sadigitID/vue-template)

## NestJS
```bash
npx degit sadigitid/nest-template nama-project
```
[sadigitID/nest-template](https://github.com/sadigitID/nest-template)
*Termasuk: modular architecture, Swagger, DTO validation, exception filter, rate limiting, testing, Docker*

---

**Catatan Penting:**
- Semua template tersebut berisi standar coding, konfigurasi, dan best practices yang telah ditetapkan oleh tim.
- Pastikan untuk membaca dokumentasi di masing-masing repository template sebelum memulai pengembangan.
- Jangan mengubah konfigurasi dasar yang telah disediakan tanpa persetujuan tim.
